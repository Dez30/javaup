## 📖 项目简介

> **JavaUp** 是一个系统化的 Java 技术学习平台，专注于 Java 八股文和实战技能的全面提升。

本知识库致力于为学习 Java 的小伙伴提供：
- ✅ **全面的知识体系**：从基础到进阶，构建完整的 Java 技术图谱
- ✅ **面试必备八股文**：精心整理各大厂高频面试题，深入浅出讲解
- ✅ **优质的学习资源**：精选书籍、视频、文档，助你快速成长
- ✅ **实战项目经验**：理论结合实践，掌握企业级开发技能

## 🚀 快速开始

### 📦 安装依赖

```bash
# 使用 yarn 安装
yarn

# 或使用 npm 安装
npm install
```

### 💻 本地开发

```bash
# 启动开发服务器
yarn start

# 或使用 npm
npm run start
```

执行命令后，会自动打开浏览器访问 `http://localhost:3000`，大部分修改都会实时热更新。

### 🔨 构建项目

```bash
# 构建生产版本
yarn build

# 或使用 npm
npm run build
```

构建完成后，静态文件会生成到 `build` 目录，可以部署到任何静态服务器。

### 🌐 部署

#### 使用 SSH 部署：

```bash
USE_SSH=true yarn deploy
```

#### 不使用 SSH 部署：

```bash
GIT_USER=<Your GitHub username> yarn deploy
```

> 如果使用 GitHub Pages 托管，此命令会自动构建网站并推送到 `gh-pages` 分支。
